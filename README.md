## Backend - .NET 8 API

### Requisitos
- .NET 8 SDK
- PostgreSQL

### Passos

1. Configurar string de conexão em appsettings.json
2. Criar banco no PostgreSQL
3. Rodar:

```bash
dotnet restore
dotnet run
```
